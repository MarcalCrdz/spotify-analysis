# 🎧 Spotify Data Analysis

Projeto de análise exploratória de dados utilizando Python para identificar padrões e insights em músicas do Spotify.

---

## 🚀 Objetivo

Explorar dados reais de músicas para entender relações entre características musicais, popularidade e gêneros, aplicando técnicas de análise de dados e visualização.

---

## 📊 Dataset

Dataset contendo informações sobre músicas do Spotify, incluindo:

* Popularidade
* Energia
* Dançabilidade
* Gênero musical
* Outras características sonoras

---

## 🧠 Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn

---

## ⚙️ Etapas do projeto

* Carregamento e tratamento de dados
* Análise exploratória (EDA)
* Identificação de valores nulos
* Visualização de dados com gráficos
* Análise de correlação entre variáveis
* Geração de insights

---

## 📈 Principais análises

### 🎵 Gêneros mais frequentes

Identificação dos gêneros musicais mais presentes no dataset.

### ⚡ Energia vs Popularidade

Análise da relação entre intensidade da música e seu nível de popularidade.

### 📊 Distribuição da popularidade

Visualização de como a popularidade está distribuída entre as músicas.

### 🔗 Correlação entre variáveis

Análise de relações entre características musicais.

---

## 💡 Insights

* Alguns gêneros aparecem com maior frequência, indicando concentração no dataset
* Músicas com maior energia tendem a apresentar maior popularidade, mas não é uma relação absoluta
* A maioria das músicas possui popularidade média
* Características musicais apresentam correlação entre si
